### الجزء الثاني: إعادة تصميم كارت الصنف (ProductVariantsModal)

#### التصميم الجديد على نمط أودو:

**1. هيدر محسّن مع شريط تقدم**
- عرض صورة المنتج والمعلومات الأساسية
- شريط تقدم بصري للمخزون الكلي
- مؤشر الحالة (متوفر/منخفض/نفد)

**2. تبويبات ذكية**
- **المخزون**: مصفوفة ثنائية (ألوان × مقاسات) مع الكميات
- **الأسعار**: مصفوفة تعرض التعديلات السعرية لكل متغير
- **الباركود/SKU**: مصفوفة تعرض الباركود و SKU

**3. مصفوفة المخزون (Stock Matrix)**
```text
┌─────────────┬──────┬──────┬──────┬──────┐
│   اللون     │  XS  │  S   │  M   │  L   │
├─────────────┼──────┼──────┼──────┼──────┤
│ ⬤ أسود     │  15  │  20  │  25  │  18  │
│ ⬤ أبيض     │   8  │  12  │  15  │  10  │
│ ⬤ أحمر     │   0  │   5  │   8  │   3  │
└─────────────┴──────┴──────┴──────┴──────┘
```

**4. ترميز الألوان للكميات**
- أخضر: كمية جيدة (> 10)
- أصفر: مخزون منخفض (1-10)
- أحمر: نفد (0)

**5. Tooltip تفاعلي**
- عند التحويم على خلية يظهر:
  - SKU الكامل
  - الباركود
  - السعر النهائي
  - التكلفة
  - آخر حركة

### التفاصيل التقنية

#### هيكل المصفوفة الجديد
```tsx
interface MatrixCell {
  colorId: string;
  sizeId: string;
  stock: number;
  sku: string;
  barcode: string;
  price: number;
  cost: number;
}

interface StockMatrixProps {
  variants: ProductVariant[];
  basePrice: number;
  baseCost: number;
  mode: 'stock' | 'price' | 'barcode';
}
```

#### تجميع البيانات للمصفوفة
- استخراج الألوان الفريدة من المتغيرات
- استخراج المقاسات الفريدة من المتغيرات
- بناء مصفوفة ثنائية الأبعاد

#### الترميز اللوني
```tsx
const getStockColor = (stock: number) => {
  if (stock === 0) return 'bg-red-100 text-red-700';
  if (stock <= 10) return 'bg-yellow-100 text-yellow-700';
  return 'bg-green-100 text-green-700';
};
```
